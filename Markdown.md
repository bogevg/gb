# Подсказки по командной строке
командасмены дириктории
```sh
cd c:\folder_name
```

Листинг текущей директории
```sh
dir
```
Создание репозитория
```sh
init
```
Удаление файла
```sh
rm
```
Отслеживать файлы
```sh
add
```
Показать все коммиты
```sh
log
```
Создать ветку
```sh
branch <name>
```
Показать все ветки
```sh
branch
```
Удалить ветку
```sh
branch -d <name>
```
Слияние веток
```sh
merge
```
Переход на ветку или коммит 
```sh
checkout
```
